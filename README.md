# Ice Realty Group - Backend Client

## Package Manager

This repo uses **Yarn** exclusively. Do not use `npm install` or commit `package-lock.json`.

## Setup

```bash
yarn install
```

## Scripts

```bash
yarn dev      # Start dev server on port 2000 (Turbopack)
yarn build    # Production build (Turbopack)
yarn start    # Start production server on port 2000
yarn lint     # Run ESLint
```
